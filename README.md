# MTProto-manager
```bash
wget -O install.sh https://raw.githubusercontent.com/androideworld/MTProto-manager/refs/heads/main/install.sh && chmod +x install.sh && sudo ./install.sh

```

# 1. Сохраните скрипт
```bash
sudo nano /usr/local/bin/mtproto-manager
```
Вставьте код выше → Ctrl+O → Enter → Ctrl+X

# 2. Сделайте исполняемым
```bash
sudo chmod +x /usr/local/bin/mtproto-manager
```
# 3. Проверка
```bash
sudo bash -n /usr/local/bin/mtproto-manager && echo "✅ Синтаксис ОК"
```

# 4. Запуск
```bash
sudo mtproto-manager
```
